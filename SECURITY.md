# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Honcho MCP Server, please report it privately by opening a security advisory at:

https://github.com/devstroop/honcho-mcp/security/advisories

Please **do not** report security vulnerabilities through public GitHub issues.

We ask that you give us a reasonable time frame to address the issue before disclosing it publicly.

## What to Include

- A description of the vulnerability
- Steps to reproduce it
- Potential impact
- Any suggested fix (if known)

## Response Timeline

- **Acknowledgement** within 48 hours
- **Root cause analysis** within 5 business days
- **Fix timeline** communicated within the analysis

## Scope

This policy covers the Honcho MCP Server codebase. For vulnerabilities in the Honcho backend itself, please report to the Honcho team directly.
